Overview

Angular Radial Color Picker has been developed treating the mobile devices as first class citizens. This means that out of the box you get:

  • Touch devices support
  • Responsive size
  • Blazing fast GPU accelerated animations
  • Super small size (~6 kB minified + gzipped)

The size is the overall module size including the Propeller dependency. For maximum browser support the color wheel is actualy a PNG file (59 kB), but we're looking into a pre-generated SVG asset or a CSS solution such as linear-gradient().

Sizing

Our color picker is in the <md-dialog> component. On select the color picker closes itself and closes the dialog.

As you might have noticed the color picker changes it's size when there is enough space. There is no voodoo magic - just plain old media queries. 

color-picker {
    width: 230px;
    height: 230px;
}
@media screen and (min-width: 840px) {
    color-picker {
        width: 350px;
        height: 350px;
    }
}

Animations

According to Google's material design guidelines desktop animations should be faster than those on mobile. The <color-picker> was written with extensibility in mind.

You can tweek the animations' duration with simple css. The example below shows the animations sped up by 30%. Wrap the rules in a @media screen and (min-width: 960px) {} and you got yourself nice and quick transitions on larger screens and slower and smooth transitions on smaller screens.

You can even go further and change the animation timing functions. Sky's the limit!

color-picker .color-palette {
    /* color wheel animation */
    transition-duration: .35s;
}
color-picker .color {
    /* selector (middle button) pulse animation */
    animation-duration: .28s;

    /* selector (middle button) appear/disappear */
    transition-duration: .49s;
}
color-picker .color-shadow-animate {
    /* ripple animation */
    animation-duration: .35s;
}
color-picker .knob {
    /* first property is for knob shadow on drag */
    /* second property is for knob appear/disappear */
    transition-duration: .2s, .28s;
}
